# Online Library Website

This project is developed for the **IS231 - Web Technology** course.

## Project Description

The Online Library Website allows users to browse books, search for books, and borrow them.  
The system supports two types of users:

- **Admin**
- **User**

Admins can manage books, while users can search and borrow books.

This is **Phase 1** of the project, which focuses on designing the website interface using **HTML only** without any frameworks.

---

## Features

### Admin
- Add new books
- View all books
- View book details
- Edit book details
- Delete books

### User
- Browse available books
- View book details
- Borrow books
- Search for books
- View borrowed books

---

## Pages

The website contains the following HTML pages:

- `index.html` – Home page
- `login.html` – User login
- `signup.html` – User registration
- `search.html` – Search for books
- `borrowed_books.html` – List of borrowed books
- `admin_books.html` – Admin book management
- `admin_book_details.html` – Admin book details
- `admin_add_book.html` – Add new book
- `admin_edit_book.html` – Edit book information
- `user_books.html` – List of books for users
- `user_book_details.html` – Book details for users
- `cover.html` – Project cover page
